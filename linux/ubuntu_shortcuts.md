## 2. 自定义快捷键
1. 定义系统监视器

该功能相当于Windows下的资源管理器`ctrl+alt+delete`
`System Settings` -->` keyboard` -->`Shortcuts `--> `Custom Shortcuts` 
点击那个“+”来添加一个快捷键。

Name: `SystemMonitor`

Command: `gnome-system-monitor`

2. 定义快速启动文件夹

 Name: `Archives`

Command: `nautilus`

3.  定义截屏

 name : `screen `
 command : ` gnome-screenshot -a `

4. 快速启动谷歌浏览器

name ：`google-chrome`
command ： `google-chrome`
